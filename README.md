# 💝 辦公室健康時鐘

一款常駐於桌面角落的辦公室健康小幫手，幫助久坐上班族養成運動、喝水、按時服藥的好習慣。

![Python](https://img.shields.io/badge/Python-3.12-blue) ![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey) ![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ 功能特色

| 功能 | 說明 |
|------|------|
| 🕐 **桌面時鐘** | 時鐘顯示於抬頭列，始終置頂，可自由拖曳 |
| 🏃 **久坐提醒** | 每 50 分鐘強制彈出提醒，必須按 OK 或 ESC 才能繼續操作 |
| 💧 **喝水追蹤** | 記錄每日飲水量，圓形進度指示器，不定期提醒補水 |
| ⏰ **下班前提醒** | 下班前 30 分鐘，若未達喝水目標自動提醒 |
| 💊 **用藥提醒** | 每日 13:40 提醒服藥，服藥後顯示「已服 ✓」，跨重開機保留記錄 |
| 🌤️ **即時天氣** | 自動偵測所在地，顯示溫度、風速、降雨（Open-Meteo API，無需 Key） |
| 🖥️ **系統監控** | CPU/GPU 溫度、CPU 忙碌度、RAM/VRAM 用量、CUDA 使用率 |
| 📐 **可調大小** | 視窗邊緣可拖曳調整大小，自動記住位置 |
| ⚙️ **彈性設定** | 下班時間、喝水目標、提醒間隔、用藥時間均可調整 |
| 🚀 **開機自動啟動** | 登入後自動在背景啟動，無需手動開啟 |

## 🚀 快速開始

### 系統需求

- Windows 10 / 11
- Python 3.10+（若系統有 `winget`，可由一鍵啟動自動安裝）

### 安裝與啟動

1. 下載或 Clone 此專案
2. 雙擊 `start.bat` 或 `一鍵啟動.bat`（第一次執行會自動偵測 Python、建立虛擬環境、安裝依賴，並加入 Windows 開機啟動）
3. 時鐘視窗出現於螢幕右下角，可拖曳到任意位置

```bash
git clone https://github.com/vincentchiou/office-health-clock.git
cd office-health-clock
start.bat
```

> **自動布署**：啟動器會自動偵測 Python（py / python / winget），建立虛擬環境，安裝依賴。可選依賴安裝失敗時主程式仍可降級運作。

## 🌤️ 天氣功能

- 自動透過 IP 偵測所在縣市
- 使用 [Open-Meteo](https://open-meteo.com/) 免費 API，無需 API Key
- 顯示：溫度、風速、降雨量
- 啟動時抓取一次，之後每 2 小時更新

## 🖥️ 系統監控

使用以下方式讀取硬體資訊：

| 指標 | 來源 |
|------|------|
| CPU 溫度 | LibreHardwareMonitor DLL（pythonnet）/ PowerShell helper |
| CPU 忙碌度 | psutil |
| GPU 溫度 | pynvml (NVIDIA) |
| GPU 使用率 | pynvml (NVIDIA) |
| RAM | psutil |
| VRAM | pynvml (NVIDIA) |
| CUDA 使用率 | pynvml (NVIDIA) |

> CPU 溫度需要管理員權限，透過背景 PowerShell helper 以 elevated 方式執行。

## ⚙️ 個人化設定

點擊視窗右上角「⚙ 設定」開啟設定對話框：

| 設定項目 | 預設值 | 說明 |
|---------|--------|------|
| 每日喝水目標 | 2000 ml | 每日建議飲水量 |
| 下班時間 | 17:30 | 超過此時間停止喝水提醒 |
| 久坐提醒間隔 | 50 分鐘 | 運動提醒頻率 |
| 喝水提醒間隔 | 30～90 分鐘（隨機）| 喝水提醒頻率範圍 |
| 用藥提醒時間 | 13:40 | 每日用藥提醒時間 |

設定檔存放於 `data/settings.json`，可直接編輯。

## 🔧 提醒機制說明

### 強制確認
每次提醒彈出時，視窗會搶佔輸入焦點，**必須按確認按鈕或 ESC 鍵**才能繼續操作滑鼠，確保提醒被真正注意到。

### 喝水提醒邏輯
- 工作時間內，每隔 30～90 分鐘（隨機）提醒一次
- 下班前 30 分鐘若未達標，觸發最終提醒
- 達到目標後停止提醒

### 用藥提醒邏輯
- 每日固定時間提醒
- 若跳過，15 分鐘後重新提醒
- 點選「已服藥完成」後，當日不再提醒，並記錄於 `water_log.json`

## 💡 為什麼需要「健康鐘」而不是「番茄鐘」？

> 別人做番茄鐘，是因為他們不容易進入心流。
> 工作狂不需要番茄鐘——但他們需要健康鐘。
>
> **知思行**：知道以後要有實際行動，你的人生才會改變。
> 平行時空理論：你的每一個選擇，都在決定你的人生走向。
>
> **動起來。**

## 🔬 國際實證醫學支持

### 久坐與代謝健康：最新研究怎麼說

**文獻來源：**
- BMJ Medicine（英國醫學期刊醫學分冊，2026 年 3 月）
- 日本消化系外科學會／肝臟學會《2026 代謝相關脂肪性肝病臨床實證指南（第三版）》

**研究摘要：**

醫學界近年正式將非酒精性脂肪肝（NAFLD）更名為「代謝相關脂肪性肝病（MASLD）」，定義為全身性代謝紊亂。最新臨床實證指出：

> 持續久坐超過 6 小時且未間斷，即使每天有固定走路量，體內慢性低度發炎因子仍會維持在高檔。

相反地：

> **每 45–60 分鐘主動站立並走動 3 分鐘**，能有效重置骨骼肌的葡萄糖轉運與脂肪酸氧化，是降低胰島素阻抗、逆轉脂肪肝、阻止肝纖維化的首要關鍵。

**這款健康時鐘的設計正是基於此原則** — 不需要高強度爆衝運動，只需打破長時間靜止坐姿，從底層細胞機制支持肝膽保護與代謝恢復。
